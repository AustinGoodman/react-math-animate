# React Math Animate

React Math Animate will be a library that creates automatic animated transitions for math expressions defined in a latex-inspired syntax. Uses MathML for math rendering and React Spring for transition animations. Only supported in Firefox.

## How to start the project
1. You must use Firefox at the moment.
2. Open your teminal in the project root.
3. cd client
4. npm install
5. npm start
6. navigate to [localhost:8080](http://localhost:8080/) in your browser

Code changes will be automatically reflected in the browser since we are using Webpack Dev Server.
